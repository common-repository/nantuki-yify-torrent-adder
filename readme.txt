Display movie information from YTS in wordpress post, it includes all the fields that are in IMDB, including screenshots of the movie and direct torrent download link. So if you are looking for a imdb plugin, this plugin is for you too.

== Description ==
This plugin helps to add movie information (from YTS) in wordpress post using shortcode [yts id="ytsmovieid"].

**NanTuki YiFy-Torrent Adder** is using YiFy API which provides information for the movies.

Thanks to YTS team, for the api and all the hard work You guys do. You guys are amazing :) .

== Installation ==
Using the Plugin Manager

1. Click Plugins
2. Click Add New
3. Search for `NanTuki YiFy-Torrent Adder`
4. Click Install
5. Click Install Now
6. Click Activate Plugin

Manually

1. Upload `NanTuki YiFy-Torrent Adder` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress


== Frequently Asked Questions ==


= How to Get YTS movie ID? =

To Find the YTS movie id login as admin

1. Goto `Settings->Search Yts ID`.
2. Enter the IMDB id of the desired movie
3. Click On `Submit`
4. You will see the YiFy Torrents, their ids,short description(size, quality, seeds, peers etc) and the shortcode generated for you.
5. You can just copy and paste the shortcode anywhere in your post.
